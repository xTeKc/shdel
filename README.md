# ShellDel
Delete shell history of your terminal.

__Usage__ : 
- `clone repo`
- `cargo build`
- `cargo run -q -- -b` to delete bash history
- `cargo run -q -- -z` to delete zsh history 

__Capabilities__ : 
- deletes data from *bash* and *zsh* terminal history files
